# 💰 SpendWise - Expense Tracker

A modern Expense Tracker web application built using **HTML**, **CSS**, and **Vanilla JavaScript**.

The application helps users manage daily expenses by allowing them to add, edit, delete, and store transactions permanently using Local Storage.

---

## 🚀 Features

- ✅ Add new transactions
- ✅ Edit existing transactions
- ✅ Delete transactions
- ✅ Local Storage persistence
- ✅ Dynamic DOM rendering
- ✅ Category-based icons
- ✅ Responsive modern UI
- ✅ Form validation
- ✅ Empty state screen

---

## 🛠️ Tech Stack

- JavaScript (ES6)
- Local Storage API
- Font Awesome
